# اپلیکیشن واکنش‌گرا

این پروژه یک اپلیکیشن ری‌اکت با پشتیبانی از PWA، تایپ‌اسکریپت، فونت فارسی وزیرمتن و بهینه‌سازی عملکرد است.

## نحوه اجرا

```bash
npm install
npm run dev
```

## دیپلوی

می‌توانید این اپ را روی Vercel یا Netlify با پشتیبانی از PWA منتشر کنید.

## ویژگی‌ها

- بارگذاری تنبل صفحات
- رابط فارسی
- فونت وزیرمتن
- پشتیبانی آفلاین
